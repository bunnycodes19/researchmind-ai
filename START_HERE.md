## ⚡ Troubleshooting

### Frontend blank/not loading?
1. Refresh browser (Ctrl+R)
2. Check F12 console for errors
3. Verify http://127.0.0.1:8000/docs works (backend)

### PDF won't upload?
1. Check file is actual PDF
2. Check file size (<100MB)
3. Check disk space
4. Look at backend terminal for errors

### Q&A returns no answer?
1. Wait for PDF processing (status = "Ready")
2. Try simpler question
3. Check Google Gemini API key is valid
4. Check backend logs

### API returns 401 error?
1. Make sure you're logged in
2. Copy token from login response
3. Use it in Swagger UI "Authorize" button
